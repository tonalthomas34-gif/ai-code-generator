# ⚡ AI Code Generator (No AI APIs)

A powerful **AI-like code generator website** built entirely with **HTML, CSS, and JavaScript**, without using any AI APIs.

This project mimics AI behavior using:

- 📦 JSON-based code datasets
- 🔍 Keyword extraction & matching
- 🧠 Snippet ranking system
- 🔗 Multi-snippet merging engine
- 🔄 Dynamic variable replacement

---

## 🚀 Features

- 📝 Natural language prompt input
- 🤖 Automatic programming language detection
- 🔍 Smart snippet search using tags
- 📊 Ranking system for best results
- 🧩 Multi-snippet code assembly
- 🔧 Dynamic variables (e.g., `{{PORT}}`, `{{DB_URL}}`)
- 📋 Copy-to-clipboard functionality
- 🌙 Clean dark UI (responsive design)

---

## 🧠 How It Works

- User enters a prompt
- System extracts keywords
- Detects programming language
- Searches dataset (JSON files)
- Ranks matching snippets
- Merges relevant code snippets
- Replaces variables dynamically
- Displays final generated code
